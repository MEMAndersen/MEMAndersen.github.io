setcon_cmm3

PURPOSE

rewriteL: Assembels the contraint matrix

SYNOPSIS

function [Ab,blc,buc,bs,bt,subi] = setcon_cmm3(nel,G,varargin)

DESCRIPTION 

 rewriteL: Assembels the contraint matrix
  FORMAT: Lr = rewriteL(L,nno)
 Input:
  nel: Number of elements
  G: The element material data structure:
   Concrete (reinforced or not)
    G.type{i} = 'RC';
    G.mat{i} =  [fc ft k Ax fYx Ay fYy Az fYz];
    G.A(:,i) = [G.mat{i}(4) G.mat{i}(6) G.mat{i}(8)];
   Steel 
    G.type{i} = 'S';
    G.mat{i} = fY;
 Optional input parameter pairs:
  'stressFac', float
 Output:
  Ab: Constrint matrix
  blc: Lower yield criteria
  buc: Upper yield criteria
  bs: Variable limits of steel
  bt: Variable limits of total stresses
  subi: subindexes for the semidefinte conic constraints
